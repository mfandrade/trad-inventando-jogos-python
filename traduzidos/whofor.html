Quem deve ler este livro?

  • Qualquer pessoa que queira aprender programação de computadores por si mesma, ainda que não tenha nenhuma experiência anterior em programação.
  • Crianças e adolescentes que queiram aprender programação de computadores desenvolvendo jogos. Crianças a partir dos 9 ou 10 anos não devem ter problemas para acompanhar este livro.
  • Adultos e professores que desejem ensinar programação a terceiros.
  • Programadores que queiram ensinar programação "real" a outros a partir de exemplos.

Este livro está disponível livremente sob uma licença Creative Commons de Atribuição/Compartilhamento Pela Mesma Licença. Você pode fazer tantas cópias deste livr quanto queira, desde que o crédito para o autor seja mantido. A linguagem de programação Python que este livro aborda também está livremente disponível em www.python.org.

Glossário

valor absoluto - A forma positiva de um valor negativo. Por exemplo, o valor absoluto de -2 é 2. O valor absoluto de um número positivo é ele mesmo.

IA - veja, Inteligência Artificial

algoritmo - uma série de instruções para se computar algo.

aplicações - Um programa que roda em um sistema operacional. Veja algo sobre, programa.

argumentos - Os valores que são passados por parâmetros à uma chamada de função.

inteligência artificial - Código ou programa que pode tomar decisões inteligentes (por exemplo, decisões de quando jogar) em resposta das ações do usuário.

arte ASCII - Usando caracteres de texto e espaços para desenhar figuras simples.

linguagem assembly - Linguagem de baixo nível. Instruções de linguagem Assembly são uma forma de traduzir a leitura humana para o código de máquina.

operador de atribuição - O sinal de igual "=". Usado para atribuir valores as variáveis.

declaração de variáveis - Uma linha de código que atribui um valor a uma variável usando o operador de atribuição. Isto é, cria uma variável quando ela é iniciada. Por exemplo: spam = 42

asterístico - O símbolo *. O asterístico é usado como um sinal de multiplicação.

operador combinado de atribuição - Os operadores +=, -=, *=,and /=. A declaração spam += 42 é equivalente a spam = spam + 42.

bloco - Um grupo de linhas de código com a mesma identação. Blocos podem conter outros blocos e muitas outras identações dentro deles.

boolean - Um tipo de dado que possui apenas dois valores, True e False.

operador booleano - Operadores booleanos podem conter and, or, e not.

declaração break - A declaração break imediatamente quebra o laço while ou for atual para a primeira linha após o fim do bloco do laço.

força bruta - Em criptografia, tentar toda chave possível em ordem para decriptar uma mensagem encriptada.

cifra de césar - Uma simples cifra de substituição em que cada símbolo é trocado por um outro.

sistema de coordenadas cartesianas - Um sistema de coordenadas usada para identificar pontos exatos em uma determinada área do espaço (assim como o monitor, ou em uma tela de um jogo). Sistema de coordenadas cartesianas geralmente possuem duas coordenadas, uma para o eixo X(isto é, o eixo esquerdo-direito horizontal) e outro para o eixo Y (isto é, o eixo acima e abaixo vertical).

case-sensitivity - Várias formas de representar o entendimento de um dado nome. Python é uma linguagem case-sensitivity, então spam, Spam, and SPAM são variáveis diferentes.

unidade central de processamento - UCP ou CPU, o processador principal que o seu computador usa para processar instruções de software.

cifra - Em criptografia, um algoritmo usado para encriptar e decriptar uma mensagem com uma certa chave.

cifra de texto - Em criptografia, a forma encriptada de uma mensagem.

comentário - Trecho do código fonte que é ignorado pelo interpretador Python. Comentários são usados para lembrar o programador sobre alguam coisa do seu código. Comentários iniciam com o símbolo # e continuam até o fim de uma linha.

propriedade comutativa - A propriedade de adição e multiplicação que descreve que a ordem dos fatores a serem adicionados ou multiplicados não importa. Por exemplo, 2 + 4 = 6, e 4 + 2 = 6. Também, 3 * 5 = 15, e 5 * 3 = 15.

operadores de comparação - Os operadores < (menor que), <= (menor ou igual que), > (maior que), >= (maior ou igual que), == (igual a), e != (diferente de.)

condição - Outro nome para expressões, uma destas existe em declarações if ou while que devem conter valores booleanos True ou False.

variáveis constantes - Variáveis cujo o valor não se altera. Variáveis constantes são constantemente usadas porque é mais fácil usar o nome da variável do que o valor que ela armazena. Como uma convenção, nomes de constantes são escritos com letras maiúsculas.

convenção - Uma maneira de excutar as boas práticas de programação, mas é muito usada para deixar as tarefas mais fáceis.

especificadores de conversão - O texto dentro de uma string pode ser usado para interpolação de strings. A conversão mais comum para especificar é %s, a qual especifica que a variável interpolada de ser convertida para uma string.

cpu - veja, Unidade Central de Processamento

criptanálise - A ciência de quebrar códigos secretos e cifras.

criptografia - A ciência de criar códigos secretos e cifras.

tipos de dado - Uma categoria de valores. Alguns tipos em Python são: strings, inteiros, listas, booleanos, e NoneType

decrementar - Diminuir um valor numérico por um.

decriptar - Converter uma mensagem encriptada em um texto legível.

declaração def - Um declaração que define uma função. A declaração def começa com a palavra-chave def seguida do nome da função e um conjunto de parâmetros dentro de parênteses, com qualquer quantidade de parâmetros delimitados por vírgulas. E terminando com o caracter de :. Por exemplo, def funcName(p1, p2):

delimitar - Separar com. Por exemplo, a string 'cats,dogs,mice' é delimitado por vírgulas.

dicionário - Uma estrutura de dados que pode armazenar outros valores. Valores são acessador por chave. Por exemplo, spam['foo'] = 42 atribui à chave 'foo' do dicionário spam o valor 42.

declaração else - Uma declaração else sempre segue uma declaração if, e o código dentro do bloco else é executado se a condição da declaração if for False.

lista vazia - A lista [], a qual não contem nenhum valor e o seu tamanho é zero. Veja algo sobre, string vazia.

string vazia - A string '', a qual não contem nenhum caracter e seu tamanho é zero. Veja algo sobre, lista vazia.

encriptar - Converter uma mensagem em uma forma que lembra caracteres embaralhados, e que não pode ser entendido exceto por alguém que conhece a cifra e a chave que foi usada para encriptar a mensagem.

caracter de escapmento - Caracteres de escamentos permitem ao programador especificar caracteres no Python que é defícil ou impossível tipá-lo no código. Todos caracteres de escape são precedidos por um \ caracter de barra invertida. Por exemplo, \n mostra um caracter de nova linha quando usa a função print.

avaliar - Estabelece uma redução da expressão à um único valor. A expressão 2 + 3 + 1 avalia-se ao valor 6.

executar - O interpretador Python executa linhas de código, pela avaliação de alguma expressão ou desempenho da tarefa que o código faz.

sair - Quando um programa termina. "Terminar" significa a mesma coisa.

expressão - Valores e chamadas de função conectadas por operadores. Expressões avaliadas podem ser reduzidas a um único valor.

editor de arquivos - Um programa usado para escrever ou modificar arquivos, incluindo arquivos do código fonte do Python. O programa IDLE possui um editor de arquivos que você pode usar para escrever seus programas.

números de ponto flutuante - Números fracionários ou decimais não são inteiros. Os números 3.5 e 42.1 and 5.0 são números de ponto flutuante.

Fluxograma - Um gráfico que mostra informações do fluxo de execução para um programa, e os principais eventos que ocorrem e em qual ordem.

Controle de fluxo de declarações - Declarações que causam a mudança do fluxo de execução, frequentemente dependendo das condições. Por exemplo, uma chamada de função envia um pedido de inicialização de uma função. Além disso, um laço causa a execução repetida de uma seção de código.

fluxo de execução - A ordem que as instruções Python são executadas. Gerealmente o interpretador Python inicia no topo da aplicação e desce executando uma linha por vez. Controle de fluxo de declarações podem mover o fluxo de execução para diferentes partes do código no programa.

função - Uma coleção de instruções para a serem executadas quando a função é chamada. Funções também possuem um valor de retorno, o qual é o valor a ser retornado pela função.

chamada de função - Um comando para passar execução ao código contido na função, também pode-se passar argumentos para tal função. Chamadas de funções avaliam o valor retornado pela função.

escopo global - O escopo de variáveis fora das funções. Instruções em escopo global não podem manipular variáveis de uma função em um escopo local.

hardware - Partes físicas de um computador que você pode tocar, como o teclado, mouse, disco rígido entre outros. Veja também, software.

linguagens de programação de alto-nível - Linguagens de programação que humanos conseguem entender, como o Python. Um interpretador consegue traduzir linguagem de alto-nível em linguagem de máquina, a qual o computador consegue entender.

IDLE - Interactive Development Enviroment (Ambiente de Desenvolvimento Interativo). IDLE permite que você crie seus programas.

E/S - Entrada/Saída. Isto é um termo usado para a referência do dado que é enviado em um programa (entrada) e sua reprodução (saída).

sequências imutáveis - Um tipo de dado que não pode adicionar ou remover os seus valores. No python, as duas sequências imutáveis são tipos string e tuplas.

estrutura de importação - Uma linha de código que contém a palavra reservada import acompanhada do nome de um módulo. Isto permite a você chamar qualquer função ou classe contida neste módulo.

incrementação - incrementa o valor de uma variável por um.

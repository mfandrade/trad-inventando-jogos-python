Capítulo 1 - Introdução

Olá!

Esse é um livro que vai lhe mostrar como fazer jogos de computador. Tudo que você precisa é um computador, um programa chamado Interpretador Python e esse livro. O programa é livre. Você pode baixá-lo de graça na internet. Esse livro irá lhe mostrar como configurar seu computador e programar alguns jogos. Uma vez que você aprenda como esses jogos funcionam, você poderá usar esse conhecimento para criar seus próprios jogos.

Quando eu era criança, eu encontrei um livro como esse que me ensinou como escrever meus primeiros programas e jogos. Era fácil e divertido. Agora como adulto, eu ainda me divirto programando computadores a trabalho e sou pago por isso. Mas mesmo que você não se torne um programador de computadores quando você crescer, programação é uma habilidade útil e divertida de se ter. (Eu ainda invento meus próprios jogos de computadores, às vezes.)

Computadores são máquinas muito úteis. No futuro, saber como programar um computador pode vir a ser tão útil quanto saber ler um livro. A boa notícia é que aprender a programar não é tão difícil quanto aprender a ler. Se você pode ler esse livro, você programar um computador.

Para dizer a um computador o que você quer que ele faça, isso é, para programar um computador, você irá precisar aprender uma linguagem de computador. Existem muitas linguagens de programação diferentes: Basic, Java, Python, Pascal, Haskell e C++ (pronunciado "cê mais mais").

O livro que eu li quando era criança me ensinou programação BASIC. Naquela época, a maioria das pessoas que começavam a aprender programação aprendiam a programar em BASIC. Mas novas linguagens de programação foram inventadas desde lá. Esse livro é sopre programação Python. Python é ainda mais fácil de aprender do que BASIC. Não apenas é mais fácil como também é uma linguagem séria e útil. Muitos adultos usam Python em seus próprios trabalhos e hobbies. É por isso que eu escolhi escrever esse livro sobre programação em Python.

Baixando e Instalando o Python

Você pode querer a ajuda de um adulto ou outra pessoa para baixar e instalar o Python.

O programa que executa os programas que você escreve é chamado um interpretador. O interpretador é um programa que executa programas escritos na linguagem Python. Esse interpretador é chamado de interpretador Python (ou, às vezes, nós simplesmente o chamamos de Python). Você pode baixar o interpretador Python desse website:

http://www.python.org

Clique no link de download no lado esquerdo da página. Na página de download, clique em 'Python 2.5.1 Windows Installer' para baixar o interpretador Python para Windows. (Se você está em um sistema operacional que não é o Windows, baixe o interpretador Python para esse sistema.) Podem existir versões mais novas no momento em que você estiver lendo esse livro. Se for o caso, você pode baixar a versão mais nova e os programas nesse livro ainda vão funcionar.

Dê um clique-duplo no arquivo python-2.5.1.msi que você acabou de baixar. O instalador Python será iniciado. Tudo que você precisa fazer no instalador é clicar no botão 'Next'. As escolhas padrão do instalador já são boas. Depois que a instalação terminar, clique em 'Finish'. Talvez você tenha que reiniciar o computador. Você deve salvar qualquer arquivo que você tenha em outros programas que estiverem rodando, depois clicar em Sim. Então você estará pronto para começar a programar!

Mas jogos não precisam ser muito complicados para serem divertidos. E diferente desses consoles, você sempre pode ter mais jogos criando eles você mesmo de graça. Tudo que você precisa é um computador e esse livro.
